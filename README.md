# 🏥 Hospital Attendance Analytics Dashboard

## 📌 Project Overview

This project analyzes hospital appointment attendance data to identify patterns behind patient no-shows and improve healthcare operational efficiency.

The dashboard was created using Microsoft Excel with Pivot Tables, Pivot Charts, Slicers, and interactive visualizations.

---

## 📊 Dashboard Features

- Patient Attendance Analysis
- No-Show Rate Monitoring
- Gender-wise Analysis
- Age Group Insights
- Scholarship Impact Analysis
- Hypertension & Diabetes Analysis
- Appointment Scheduling Trends
- Interactive Excel Dashboard
- KPI Summary Cards
- Dynamic Filters using Slicers

---

## 🛠️ Tools & Technologies Used

- Microsoft Excel
  - Pivot Tables
  - Pivot Charts

## 📂 Dataset Information

The dataset contains hospital appointment records including:

| Column | Description |
|---|---|
| PatientId | Unique patient identifier |
| AppointmentID | Appointment record ID |
| Gender | Male/Female |
| ScheduledDay | Appointment booking date |
| AppointmentDay | Actual appointment date |
| Age | Patient age |
| Neighbourhood | Patient area |
| Scholarship | Government welfare support |
| Hipertension | Hypertension status |
| Diabetes | Diabetes status |
| Alcoholism | Alcoholism status |
| SMS_received | SMS reminder received |
| No-show | Patient attended or not |

---

## 📈 Key Insights

- Identified factors affecting patient no-shows
- Higher no-show patterns observed in specific age groups
- SMS reminders showed impact on attendance
- Certain neighborhoods had higher absentee rates
- Health conditions influenced appointment attendance behavior
